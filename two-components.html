Two components

Here's where things get interesting. We're using exactly the same container/component as before as a notifier, but have added a second component to the page. Check out two-components.js, note that they share a common store. We might be creating two different components, but they're reacting to the same store so their representation of the total is the same. Also see that we can alter the presentation based on the value.
